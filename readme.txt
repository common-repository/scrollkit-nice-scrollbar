== Installation ==
 Download and Install. Then Use it. Easy :D 
 
1. Upload `scroll-kit.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Scroll Kit Menu 
4. Change Color with your's
5. Use it and Enjoy.
